Getting Started

How to start using jsGrid

1. Download jsGrid

Download jsGrid
combined Download source
.zip file Download source
.tar.gz file

2. Ensure that jQuery v1.8.3 or later is included.

3. Include jsGrid script and css style sheet files into the web page:

<link rel="stylesheet" href="jsgrid.min.css">
<script type='text/javascript' src='jsgrid.min.js'></script>

4. Create grid applying jQuery plugin jsGrid with grid config as follows:

$("#jsGrid").jsGrid({
    width: "100%",
    height: "400px",

    filtering: true,
    editing: true,
    sorting: true,
    paging: true,

    data: db.clients,

    fields: [
        { name: "Name", type: "text", width: 150 },
        { name: "Age", type: "number", width: 50 },
        { name: "Address", type: "text", width: 200 },
        { name: "Country", type: "select", items: db.countries, valueField: "Id", textField: "Name" },
        { name: "Married", type: "checkbox", title: "Is Married", sorting: false },
        { type: "control" }
    ]
});
